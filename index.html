How signal quality is calculated
For each managed account, we calculate the Pearson correlation (r) between each conversion action's monthly volume and verified matchback appointments (Invoca + Vetstoria). r = 1.0 = perfect predictor · r = −1.0 = perfect anti-predictor · r < 0 = remove from primary goals immediately. The recommended action set is the combination that best predicts matchback when summed.

Budget scoring model
Guardrails: Every account stays between $550 (floor) and $1,800 (ceiling) regardless of score. No account can be recommended below $550 or above $1,800. The mini bar on each row shows where the recommendation sits within the $550–$1,800 range. Score: Signal r ≥ 0.7 (+3), ≥ 0.5 (+2), ≥ 0.3 (+1), negative (−2) · MB up 2 consecutive months (+3), net up (+1), 2 down (−1) · CPA improving both months (+2), one (+1), both worse (−1) · CPA >$600 (−1), >$800 (−1 more). Uses last 3 months with data — not fixed months. Allocation: Surplus pool = cap − sum of floors. Distributed by score × tier weight (Elite 1.3×, High 1.15×, Mid 1.0×, Floor 0.85×).
